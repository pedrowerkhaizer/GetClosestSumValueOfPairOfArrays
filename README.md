# GetClosestSumValueOfPairOfArrays

So, the test consists in picking 2 random arrays and findind the closest values of theirs sum

# Example

Array 1: 6, 8, 2, 0, 5
Array 2: 0, 2, 7, 5, 3
Target Value: 7

Possible sum: {0, 7}, {6, 0}, {8, 0}, {2, 5}, {5, 3}
